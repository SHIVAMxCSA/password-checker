# Password Strength Checker 

A Python tool that checks how strong a password is and gives tips to improve it.

## Features
- Checks length, uppercase, lowercase, numbers and special characters
- Detects common weak passwords
- Gives a strength score out of 6
- Provides improvement suggestions

## Usage
```bash
python password_checker.py
```

## Example Output
Strength : STRONG ✅
Score    : 6/6
## Tech
- Python 3
- Re (regex) module
